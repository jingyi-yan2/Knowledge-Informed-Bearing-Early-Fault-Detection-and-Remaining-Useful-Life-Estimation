# Knowledge-Informed-Bearing-Early-Fault-Detection-and-Remaining-Useful-Life-Estimation
A warm-start strategy is adopted for the AE-based EFD model. A self-informed loss function, MXWSE, is introduced to guide the training of the RUL prediction model. A knowledge-informed input configuration is incorporated based on prior research to improve the AE model’s effectiveness and interpretability.
